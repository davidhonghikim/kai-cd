## Development

### Prerequisites
- Node.js 18+
- Chrome browser
- Git

### Installation
    ```bash
# Clone the repository
git clone https://github.com/davidhonghikim/chatdemon.git

# Install dependencies
    npm install

# Start development server
    npm run dev
    ```

### Building
```bash
# Build for production
npm run build
```
